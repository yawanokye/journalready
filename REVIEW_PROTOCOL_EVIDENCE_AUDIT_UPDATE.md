# ArticleReady AI 1.9.0 update

## Review protocol and evidence-audit builder

- Added structured review-method inputs for systematic, scoping, conceptual, integrative and bibliometric articles.
- Added a separate review protocol and evidence-audit output with Copy and DOCX export.
- Records formal databases, exact search strings, search dates and limits, eligibility, screening, appraisal, citation tracking, duplicate removal, synthesis method, software and final corpus information.
- Added verified record-flow fields with arithmetic consistency checks.
- Keeps ArticleReady metadata discovery separate from formal systematic-review database searching and PRISMA counts.
- Prevents fabricated databases, search dates, reviewer numbers, screening decisions, counts, corpus sizes, quality scores and bibliometric settings.
- Preserves integrative theory-building positioning for conceptual papers unless the user explicitly documents a genuine systematic or scoping protocol.
- Passes the completed protocol package into the article-drafting prompt so confirmed details replace broad author-action placeholders, while only genuinely missing items remain as focused red actions.
