Plan rules

One successful generation

Each paid package includes one successful generation and one DOCX export within the selected plan scope.

Upgrade for longer work

If a manuscript exceeds the package word, source or token allowance, the user should upgrade before continuing.

Author verification required

Data sources, instruments, permissions, ethics, results and references must still be checked by the author before submission.

No acceptance guarantee

The system improves article readiness, but journal acceptance depends on originality, evidence, fit and peer review.
