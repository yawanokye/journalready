# ArticleReady AI naming update

The application has been renamed from its previous product name to **ArticleReady AI**.

Updated areas include:

- application title and health response
- homepage, Article Topic Ideas page and Article Writer page
- navigation brand initials, now **AR**
- API documentation tags and AI system prompts
- scholarly-search user agent
- browser source-bank storage keys
- Render service name, now `articleready-ai`
- repository root folder, now `articleready-main`
- configuration variables, now prefixed with `ARTICLEREADY_`
- README and deployment documentation

DeepSeek V4 Pro remains limited to Article Topic Ideas. The Article Writer continues to use OpenAI.
